# AG2 Multi-Agent Project

| 项目 | 内容 |
|------|------|

| GitHub | https://github.com/dark-077/AG2-multiagent |

## 🎯 项目目标

基于 AG2 框架，构建一个多智能体协作系统，展示多个 AI Agent 协同完成复杂任务的能力。

## 🤖 技术栈

- **AG2**: 开源多智能体框架
- **Python 3.11+**: 编程语言
- **LLM**: 大语言模型支持 (GPT-4o / Claude / Gemini)

## 📁 项目结构

```
AG2-multiagent/
├── README.md              # 项目说明
├── requirements.txt      # 依赖包
├── hello_multiagent.py   # 基础双智能体示例
├── agent_team.py         # 多智能体团队示例
└── .env                  # API 密钥配置
```

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 API 密钥

在 `.env` 文件中添加你的 LLM API 密钥：

```env
OPENAI_API_KEY=your-api-key-here
```

### 3. 运行示例

```bash
python hello_multiagent.py
```
