## 🎬 录制内容指南

### 第 1 部分：项目介绍（15秒）

```
"大家好，我是黄荣康，这是我基于 AG2 框架构建的多智能体协作系统。
这个项目是 C5-AG2 多智能体挑战的产物。"
```

### 第 2 部分：架构展示（30秒）

**展示内容**：
1. Lead Agent 的角色和功能
2. Critic Agent 的角色和功能
3. 两者如何协作

**话术**：
```
"系统包含两个智能体：
- Lead Agent 是任务拆解专家，负责分析和分解任务
- Critic Agent 是质量审查专家，负责评估和改进方案

它们通过 GroupChat 协作完成任务。"
```

### 第 3 部分：运行演示（30秒）

1. 打开终端（建议使用 Windows Terminal 或 VS Code 终端）
2. 先演示 GroupChat 模式：
   ```bash
   python agent_team.py --mode groupchat
   ```
3. 再演示 Beta Agent-as-tool 模式（展示 `--mode beta` 新特性）：
   ```bash
   python agent_team.py --mode beta
   ```
4. 展示多智能体对话过程
5. 展示最终结果

**话术**：
```
"现在让我演示一下系统的运行效果。
[运行 --mode groupchat]
可以看到，两个智能体通过 GroupChat 平等协作...
[等待对话完成]

接下来切换 Beta 模式：
[运行 --mode beta]
这次 Critic Agent 作为 Lead 的工具被调用，展示 AG2 Beta 原生的 Agent-as-tool 模式。
[等待对话完成]

两种模式展示了 AG2 框架在不同协作场景下的灵活性。"
```

### 第 4 部分：亮点总结（15秒）

```
"项目亮点：
1. 双模式运行：同时支持 GroupChat 和 Beta Agent-as-tool
2. 国产 API 兼容：无缝接入硅基流动
3. 5 分钟上手：clone 即用

更多信息请访问 GitHub 仓库。"
```

## 🎯 录制小技巧

1. **语速适中**：每分钟 150-200 字
2. **吐字清晰**：确保麦克风收音清晰
3. **突出重点**：重点展示多智能体协作过程
4. **控制时长**：60-90 秒，不要太长
5. **多次录制**：不满意就重新录

## ✅ 提交确认

录制完成后：

1. 将视频保存为 `黄荣康_C5-AG2_demo.mp4`
2. 上传到适当平台（百度网盘/Google Drive 等）
3. 在 `黄荣康_C5-AG2_repo.md` 中添加视频链接

## 📞 遇到问题？

- 录屏黑屏？→ 检查显卡驱动和录屏软件设置
- 声音太小？→ 靠近麦克风或使用外接麦克风
- API 调用失败？→ 检查 .env 中的密钥是否正确
